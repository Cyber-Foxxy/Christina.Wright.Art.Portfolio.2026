Sky Whale Chronicles Page Upload Instructions
=============================================

This folder is ready for your main GitHub repository.

Upload these into the root of your repository:
- sky-whale.html
- sky-whale.css
- sky-whale.js
- Clockwork Dreams.mp3 or the included mp3 file
- images/sky-whale/ folder

Homepage link check:
Your top navigation and homepage carousel should link to:

<a href="sky-whale.html">Sky Whale</a>

For the carousel thumbnail, use:

<img src="images/sky-whale/sky-whale-thumbnail.jpg" alt="The Sky Whale Chronicles thumbnail" />

This page includes 62 renamed/optimized Sky Whale images.
The music button uses localStorage so it remembers the saved time and play/pause choice between pages after the visitor clicks Play once.
