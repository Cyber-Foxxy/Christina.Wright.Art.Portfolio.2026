Christina Wright Art Portfolio — Paintings Page Ready Zip

How to use:
1. Unzip this folder.
2. Upload everything inside the folder to your GitHub repository.
3. Keep this structure exactly:
   index.html
   paintings.html
   style.css
   script.js
   images/paintings/painting-01.jpg through painting-18.jpg

Working links included:
- The main page category carousel links to paintings.html
- The top navigation on index.html links to paintings.html
- The top navigation on paintings.html links back to index.html and to paintings.html

Important:
- GitHub Pages is case-sensitive. Do not rename the images unless you also update the matching HTML paths.
- The image folder must be: images/paintings/
